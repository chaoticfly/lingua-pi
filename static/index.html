Install Speech Voices

To hear authentic pronunciations in languages like Kannada, Telugu, German, or Italian, you need to install the corresponding language voice pack on your device:

Windows 10 / 11
  1. Open Settings (Win + I).
  2. Go to Time & Language > Speech.
  3. Under "Manage voices", click Add voices.
  4. Search for your language (e.g. "Kannada"), check the box, and click Add.
  5. Restart your web browser to load the new voice.
Android
  1. Open Settings.
  2. Go to System > Languages & Input > Text-to-speech output.
  3. Tap the Settings gear icon next to "Preferred engine" (usually Google TTS).
  4. Select Install voice data and download your language.
macOS
  1. Open System Settings.
  2. Go to Accessibility > Spoken Content.
  3. Select the System Voice dropdown and click Manage Voices....
  4. Find your language, click download, and click OK.
iOS (iPhone & iPad)
  1. Open Settings.
  2. Go to Accessibility > Spoken Content > Voices.
  3. Select the target language, choose a specific voice, and tap the download icon.
Linux (Raspberry Pi / Desktop)

Install system-wide speech packages via terminal, then restart the browser/server:

sudo apt-get update
sudo apt-get install espeak speech-dispatcher-espeak festival
